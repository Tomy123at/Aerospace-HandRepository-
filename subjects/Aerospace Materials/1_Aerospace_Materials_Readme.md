# Aerospace Materials

This directory contains resources focused on Aerospace Materials, deals with the study, selection, and application of materials for aircraft and spacecraft.

#### Contents:

* **Lecture Notes:** In-depth notes covering the properties, processing, and performance of various aerospace materials.
* **Lab Reports:** Examples of material testing lab reports, such as tensile tests, hardness tests, and fatigue tests
* **Case Studies:** Analysis of material selection and failure in real-world aerospace applications.
